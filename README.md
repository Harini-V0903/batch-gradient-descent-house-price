# batch-gradient-descent-house-price
---
## House Price Prediction using Batch Gradient Descent ##

## Overview ##

    This project implements Batch Gradient Descent from scratch to predict house prices using numerical input data. 
    It focuses on understanding how a machine learning model learns by iteratively updating weights and bias to minimize error. 
    The project also includes a Streamlit web application for real-time predictions.

---

## Features ##
    Implementation of Batch Gradient Descent from scratch
    Visualization of cost reduction over epochs
    Use of Mean Squared Error (MSE) as loss function
    Data scaling for improved model performance
    Interactive web app using Streamlit
    Real-time house price prediction

---

## Tech Stack ##
    Python
    NumPy
    Matplotlib
    Streamlit
    Scikit-learn

---

## Output ##
    Predicts house prices based on input features
    Displays results in real time through the web app
    Shows gradual decrease in cost during training
